# Hansard website

## Description
This website is a personal project based on my recent exploration with the UK Government API. I want to create various features to interact with the GOV API and in the process create something useful and also learn some web app skills (frontend/ backend)

### User Stories

What is the word count for X word in the past year?
What Debates has Y been in, in the past year?
Who was in a specific debate?
What did Y vote for in a debate with Z motion?
A visual of the house of commons, voting and profiles.
