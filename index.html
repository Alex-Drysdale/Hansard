Welcome to Hansard

What is Hansard?

  • Hansard is a project by Xandi Drysdale that uses the UK Gov API
  • It is a project to practice making a backend for a web app
  • It answers the following user stories
    1. A profile of member Y
    2. What debates has Y been in, in the past year?
    3. Who was in a specific debate?
    4. What did Y vote for in a debate with Z motion?
    5. A visual of the house of commons, voting and profiles.
